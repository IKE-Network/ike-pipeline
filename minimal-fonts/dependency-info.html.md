# Maven Coordinates

## [Apache Maven](#apache-maven)

```
<dependency>
  <groupId>network.ike</groupId>
  <artifactId>minimal-fonts</artifactId>
  <version>55</version>
</dependency>
```

## [Apache Ivy](#apache-ivy)

```
<dependency org="network.ike" name="minimal-fonts" rev="55">
  <artifact name="minimal-fonts" type="jar" />
</dependency>
```

## [Groovy Grape](#groovy-grape)

```
@Grapes(
@Grab(group='network.ike', module='minimal-fonts', version='55')
)
```

## [Gradle/Grails](#gradle-grails)

```
implementation 'network.ike:minimal-fonts:55'
```

## [Scala SBT](#scala-sbt)

```
libraryDependencies += "network.ike" % "minimal-fonts" % "55"
```

## [Leiningen](#leiningen)

```
[network.ike/minimal-fonts "55"]
```
