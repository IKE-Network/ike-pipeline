=== Project Structure

The project follows Maven standard directory layout:

[source]
----
example-project/
├── src/
│   ├── main/
│   │   ├── java/               # Java source code
│   │   │   └── org/ike/example/
│   │   │       └── Concept.java
│   │   └── resources/          # Application resources
│   ├── test/
│   │   ├── java/               # Test source code
│   │   │   └── org/ike/example/
│   │   │       └── ConceptTest.java
│   │   └── resources/          # Test resources
│   └── docs/
│       ├── asciidoc/           # Documentation source
│       │   ├── index.adoc
│       │   └── chapters/       # Modular documentation
│       └── resources/          # Images, etc.
├── pom.xml
└── target/
    ├── classes/                # Compiled Java
    ├── test-classes/           # Compiled tests
    ├── generated-sources/      # Unpacked doc dependencies
    │   └── asciidoc/
    └── generated-docs/         # Generated documentation
        ├── html/
        └── pdf/
----

=== Build Lifecycle

The Maven build follows this sequence:

[plantuml]
....
@startuml
start

:generate-sources|
note right
  Unpack doc dependencies
  to target/generated-sources/
end note

:compile|
note right
  Compile Java 25 source
  with preview features
end note

:test|
note right
  Run unit tests with
  JUnit 5 + AssertJ
end note

:package|
note right
  Create JAR with
  manifest entries
end note

:verify|
note right
  Run integration tests
  Generate documentation
end note

:install|
note right
  Install to local
  Maven repository
end note

stop
@enduml
....
