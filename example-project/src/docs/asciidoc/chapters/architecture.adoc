=== Project Structure

The project follows Maven standard directory layout:

[source]
----
example-project/
├── src/
│   ├── main/
│   │   └── java/               # Java source code
│   │       └── org/ike/example/
│   │           └── Concept.java
│   ├── test/
│   │   └── java/               # Test source code
│   │       └── org/ike/example/
│   │           └── ConceptTest.java
│   └── docs/
│       └── asciidoc/           # Documentation source
│           ├── index.adoc
│           └── chapters/
├── pom.xml
└── target/
    ├── classes/                # Compiled Java
    ├── test-classes/           # Compiled tests
    └── generated-docs/         # Generated documentation
        ├── html/
        └── pdf-{renderer}/
----

=== Maven Parent Hierarchy

The project inherits from IKE parent POMs:

[plantuml]
....
@startuml
skinparam rectangleBorderColor #999999
skinparam arrowColor #666666
skinparam defaultFontName Noto Sans

rectangle "ike-parent" as A #e1f5ff
rectangle "java-parent" as B #fff4e1
rectangle "example-project" as C #e8f5e9

A --> B
B --> C

A ..> "AsciiDoc Pipeline" : provides
B ..> "Java 25 Config" : provides
B ..> "Test Frameworks" : provides
C ..> "JAR + Docs" : produces
@enduml
....
