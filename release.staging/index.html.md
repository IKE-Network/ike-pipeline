# About IKE Pipeline

Reactor root and build infrastructure for the IKE Community. Provides the AsciiDoc documentation pipeline (6 PDF renderers), Java 25 compiler/test configuration, GPG signing, and shared plugin management. External projects inherit ike-parent (build conventions and centralized dependency versions).
