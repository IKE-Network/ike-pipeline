= IKE Parent POM

Centralized dependency version management and AsciiDoc toolchain
configuration for IKE Community projects.
Inherit from `ike-parent` to get aligned artifact versions
and the full multi-renderer documentation pipeline
without declaring explicit versions on each dependency.

== Usage

Set `ike-parent` as your project's parent:

[source,xml]
----
<parent>
    <groupId>network.ike</groupId>
    <artifactId>ike-parent</artifactId>
    <version>${ike.pipeline.version}</version>
</parent>
----

Then declare dependencies without version:

[source,xml]
----
<dependency>
    <groupId>network.ike</groupId>
    <artifactId>koncept-asciidoc-extension</artifactId>
</dependency>
----

== Managed Artifacts

`ike-parent` manages versions for all IKE pipeline artifacts inline
via its `<dependencyManagement>` section:
`ike-doc-resources`,
`minimal-fonts`,
`docbook-xsl`,
`koncept-asciidoc-extension`,
`semantic-linebreak`,
and `ike-build-tools`.

All artifacts share the unified pipeline version.
