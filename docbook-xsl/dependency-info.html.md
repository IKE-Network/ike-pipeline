# Maven Coordinates

## [Apache Maven](#apache-maven)

```
<dependency>
  <groupId>network.ike</groupId>
  <artifactId>docbook-xsl</artifactId>
  <version>55</version>
</dependency>
```

## [Apache Ivy](#apache-ivy)

```
<dependency org="network.ike" name="docbook-xsl" rev="55">
  <artifact name="docbook-xsl" type="jar" />
</dependency>
```

## [Groovy Grape](#groovy-grape)

```
@Grapes(
@Grab(group='network.ike', module='docbook-xsl', version='55')
)
```

## [Gradle/Grails](#gradle-grails)

```
implementation 'network.ike:docbook-xsl:55'
```

## [Scala SBT](#scala-sbt)

```
libraryDependencies += "network.ike" % "docbook-xsl" % "55"
```

## [Leiningen](#leiningen)

```
[network.ike/docbook-xsl "55"]
```
