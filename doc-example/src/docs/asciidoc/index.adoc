== Introduction

The IKE documentation pipeline converts AsciiDoc source to:

* **HTML5** — multi-page with table of contents
* **Self-contained HTML** — single file with embedded images
* **PDF via Prawn** — built-in, free (asciidoctorj-pdf)
* **PDF via FOP** — free, pure Java (DocBook -> XSL-FO -> PDF)
* **PDF via Prince XML** — commercial CSS Paged Media
* **PDF via Antenna House** — commercial CSS Paged Media
* **PDF via WeasyPrint** — free Python-based CSS Paged Media
* **PDF via RenderX XEP** — DocBook -> XSL-FO -> PDF

This document tests each feature across all renderers.

== Build Reference

=== Quick Start

[source,bash]
----
# HTML only (default):
mvn clean verify

# HTML + Prawn PDF:
mvn clean verify -Dike.pdf.prawn

# HTML + FOP PDF:
mvn clean verify -Dike.pdf.fop

# Multiple renderers:
mvn clean verify -Dike.pdf.prawn -Dike.pdf.fop
----
