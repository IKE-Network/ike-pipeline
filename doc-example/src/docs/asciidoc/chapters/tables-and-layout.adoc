=== Sidebar

.Pipeline Architecture
****
The IKE documentation pipeline is built on Maven 4's reactor model.
Infrastructure modules (fonts, themes, extensions) are installed first,
then child projects inherit the full toolchain via parent POM inheritance.
****

=== Page Breaks

In PDF output, a new chapter (level-2 heading) forces a page break.
This is controlled by the CSS rule `h2 { page-break-before: always; }`
in the print stylesheet, and by the Prawn theme's heading configuration.

=== Images

The pipeline supports inline SVG diagrams (rendered by Kroki) and
raster images. SVG is preferred for scalability and print quality.
