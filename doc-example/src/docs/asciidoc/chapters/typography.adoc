=== Font Families

The IKE pipeline ships the Noto font family for consistent cross-platform rendering:

Serif (body text):: Noto Serif — used for paragraph text throughout.

Sans-serif (headings):: Noto Sans — used for headings, table headers, and running headers/footers.

Monospace (code):: Noto Sans Mono — used for code blocks and inline code.

=== Unicode Symbols

Mathematical operators (Noto Sans Math):

* Subsumption: &#x2291; (is-a)
* Intersection: &#x2293; (and)
* Existential: &#x2203; (some)
* Universal: &#x2200; (all)
* Equivalence: &#x2261; (defined-as)
* Union: &#x2294; (or)
* Negation: &#x00AC; (not)

Technical symbols (Noto Sans Symbols):

* Arrows: &#x2190; &#x2191; &#x2192; &#x2193; &#x2194; &#x2195;
* Check mark: &#x2713;
* Cross mark: &#x2717;
* Section: &#x00A7;
* Paragraph: &#x00B6;

=== Code Blocks

Java example with syntax highlighting:

[source,java]
----
public record DocumentMetadata(
    String title,
    String author,
    LocalDate publishDate,
    List<String> keywords
) {
    public DocumentMetadata {
        Objects.requireNonNull(title, "title");
        Objects.requireNonNull(author, "author");
        keywords = List.copyOf(keywords);
    }
}
----
