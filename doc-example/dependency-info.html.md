# Maven Coordinates

## [Apache Maven](#apache-maven)

```
<dependency>
  <groupId>network.ike</groupId>
  <artifactId>doc-example</artifactId>
  <version>55</version>
  <type>pom</type>
</dependency>
```

## [Apache Ivy](#apache-ivy)

```
<dependency org="network.ike" name="doc-example" rev="55">
  <artifact name="doc-example" type="pom" />
</dependency>
```

## [Groovy Grape](#groovy-grape)

```
@Grapes(
@Grab(group='network.ike', module='doc-example', version='55')
)
```

## [Gradle/Grails](#gradle-grails)

```
implementation 'network.ike:doc-example:55'
```

## [Scala SBT](#scala-sbt)

```
libraryDependencies += "network.ike" % "doc-example" % "55"
```

## [Leiningen](#leiningen)

```
[network.ike/doc-example "55"]
```
