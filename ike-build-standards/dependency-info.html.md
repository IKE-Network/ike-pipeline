# Maven Coordinates

```
<dependency>
  <groupId>network.ike</groupId>
  <artifactId>ike-build-standards</artifactId>
  <version>49</version>
  <type>pom</type>
</dependency>
```

## [Apache Ivy](#apache-ivy)

```
<dependency org="network.ike" name="ike-build-standards" rev="49">

</dependency>
```

